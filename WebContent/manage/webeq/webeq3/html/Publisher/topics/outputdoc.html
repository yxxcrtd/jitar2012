The Output Document

The main window of WebEQ Publisher is divided into two panels. You specify where the output should go in the bottom panel. You also specify the output format for equations in the bottom panel as well.

Setting up the Output

The Output Document panel at the top of the main window contains two fields for specifying what WebEQ Publisher should do with output:

Output to File
When checked, this directs the Publisher to send output to the named file specified in the adjacent input area. You can use the file browser to pick an output file as well.
Output to Screen
When checked, output is directed to a re-sizeable pop-up window.
