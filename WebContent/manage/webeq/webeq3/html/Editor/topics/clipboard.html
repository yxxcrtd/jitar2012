Viewing the System Clipboard

Every cut or copy operation in WebEQ Editor places data on the system clipboard. By default, a presentation MathML encoding of the current selection is placed on the clipboard. Using the Copy Special options, you can place WebEQ Viewer Control applet code or an HTML image tag on the clipboard. You can also generate several kinds of MathML markup.

Since choosing between different clipboard formats can be confusing, the Show System Clipboard option displays the data on the system clipboard, so you can verify the you have what you were expecting. Note that this is also a handy way to directly inspect the MathML structure of the current equation.
