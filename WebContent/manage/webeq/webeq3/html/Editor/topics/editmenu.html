Cutting and Pasting

The Edit Menu provides cut and paste related operations.

Selection

To cut or copy, you must first select an expression. WebEQ Editor uses structured selection to insure the MathML structure of the equation is preserved. This means that you cannot select part of one template and part of another. In practical terms, as you drag the cursor to sweep out a selection, when you cross over a template boundary, the entire template will be selected.

Another convenient way to make selections is via the MathML Ancestry. Clicking on the name of a template in the Ancestry will select it in the editor.

One final way to select an expression is with the backspace and delete keys. Ordinarily, backspace and delete remove the character to the left and right of the cursor respectively. However, if the object adjacent to the cursor is a nested template, hitting these keys will first highlight the expression. This give you a chance to reconsider doing the deletion, in case you didn't really mean to delete the entire template. Hitting backspace or delete a second time will then remove the highlighted expression as you would expect.

The Edit menu gives you the following choices:

Undo
This choice undoes the last cut or paste operation. WebEQ Editor remembers the last three operations so it is possible to undo up to three operations. The keyboard shortcut for Undo is ctrl-z.
Cut
This removes the current selection from the equation, and places a presentation MathML representation of the selection on the system clipboard. The keyboard shortcut for Cut is ctrl-x.
Copy
This places a presentation MathML representation of the current selection on the system clipboard without removing the selection from the current equation. The keyboard shortcut for Copy is ctrl-c.
Copy Special
The Copy Special submenu gives you three choices for the format to use when putting equation data on the system clipboard. It also lets you copy three useful kinds of document header declarations to the clipboard. For details see Copy As Special.
Paste
This option pastes the equation fragment currently on the clipboard back into the current equation at the current cursor location. WebEQ Editor can paste anything it places on the clipboard back into the equation, regardles of the format of the clipboard data. However, currently you cannot paste data other applications put on the clipboard into your equation. The keyboard shortcut for Paste is ctrl-v.
Show System Clipboard
This option displays a text representation of whatever is currently on the system clipboard. For more information about this option, see the section devoted to Viewing the System Clipboard
