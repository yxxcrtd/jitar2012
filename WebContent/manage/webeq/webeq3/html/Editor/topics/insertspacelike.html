Inserting Space-like Templates

The following templates can be inserted from the Space-like Layout submenu of the Insert Menu

  • MSpace inserts an mspace element

    This also brings up a dialog for changing the default attributes of the mspace:

    The fields in this dialog correspond to attributes of the mspace element. Each field accepts one MathML size value. Note that he default value is 0 for all three attributes, so even though a percentage is an admissible value, it will have no effect.

  • MPhantom inserts an mphantom template

    The mphantom element represents a space of the same dimensions as the expression it wraps.
