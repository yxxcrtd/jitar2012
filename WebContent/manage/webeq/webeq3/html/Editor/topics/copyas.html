Copy Special

The Copy Special submenu gives three options for putting the current equation on the system clipboard in different formats. There are also three options for copying header declarations to the clipboard.

The three format choices are:

Copy HTML Applet Tag
This option places Viewer Control applet code for displaying the current selection on the system clipboard. When choosing this option, a dialog appears reviewing current Applet Export Preferences, and giving you a chance to accept or edit them.
Copy HTML Image Tag
This option places an HTML image tag on the system clipboard. This option only works if you have first saved the equation as a image. See the information on the File Menu for details on saving images. When choosing this option, a dialog appears reviewing current Image Export Preferences, giving you a chance to accept or edit them.
Copy MathML
This option places MathML markup on the system clipboard. Since presentation MathML is the default format for clipboard data, this is primarily useful when you want to generate MathML with special options set, such as a namespace prefix, or when you want to generate content markup instead of presentation markup. When choosing this option, a dialog appears reviewing current Mathml Export Preferences, giving you a chance to accept or edit them.

The header declaration choices are:

Copy MathPlayer Declarations
This option places the header declarations on the clipboard that cause MathML content in a Web page to render with Design Science's MathPlayer behavior for Internet Explorer.
Copy Math Stylesheet Declaration
This option places header declarations on the clipboard that specify MathML rendering be dynamically determined by the Universal Math XSL Stylesheet. This XSL stylesheet determines what MathML rendering options are available on a client browser, and transforms the document accordingly. This allows authors to write a single document which works with IE6 + MathPlayer, IE6 + Techexplorer, Mozilla, and several other platforms.
Copy Mozilla Declaration
This options places header declarations on the clipboard for XHTML documents to be rendered with the Mozilla browser.
