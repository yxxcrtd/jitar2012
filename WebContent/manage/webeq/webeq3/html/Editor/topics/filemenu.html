Saving and Opening Files

Saving and opening files is accomplished through the File Menu.

The four basic options on the File menu are:

New
Selecting New clears the current equation and starts a new blank equation. If you have modified the current equation since the last save, you will be asked whether you wish to save your changes.
Open MathML
This option loads a saved equation into the Editor. The editor can open any valid MathML equation. When you open an equation encoded as content MathML in WebEQ Editor, the visual appearance may be slightly different than you are expecting, since content MathML only encodes the meaning of an equation. In some instances, MathML created by third-party tools may not open correctly because of XML directives that WebEQ Editor doesn't understand, such as processing instructions and doctype declarations. In these cases, remove the additional XML directives with a text editor before opening the equation with WebEQ Editor.
Save
After creating a new equation, the Save option is initially disabled. For the first Save, you must choose the format for the output by using Save As HTML Applet, Save As Image, or Save As MathML. After the first save, however, the Save item will become active, and its label will change to reflect the format you've choosen. Selecting it from the menu will continue to save the equation in the last selected format until you change the format.
Quit
This option closes WebEQ Editor. You will be prompted to save your changes if you have edited the current equation since the last save.
