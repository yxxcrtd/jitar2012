Keyboard Shortcuts

Though most people prefer to use menus and a mouse when first learning, it is convenient to have keyboard shortcuts for very common operations. The WebEQ Input Control has a number of useful shortcuts, which are listed below.

Editing Shortcuts

One set of keyboard shortcuts handle the main editing functions. These shortcuts all use the control key together with a letter:

Shortcut Key Action Supported Platforms
ctrl-a select all all
ctrl-c copy all
ctrl-v paste all
ctrl-x cut all
ctrl-z undo all
ctrl-, shrink not Mac OSX
ctrl-. magnify not Mac OSX

Template Shortcuts

A second set of keyboard shortcuts handle the insertion of templates. These shortcuts all use the control key together with a letter:

Shortcut Key Action Supported Platforms
ctrl-r insert <mrow> all
ctrl-f insert <mfrac> all
ctrl-/ insert <mfrac> not Mac OSX
ctrl-b or ctrl-_ insert <msub> all
ctrl-p or ctrl-^ insert <msup> all
ctrl-Q insert <msqrt> all
ctrl-R insert <mroot> all
ctrl-P insert parenthesized expression all

In addition, the TAB key cycles the cursor between the open template blanks in an equation, and the '^' character (by itself without the CTRL key) adds a superscript to the previous expression so that typing 'x^2' gives x with a superscript as expected.
