Saving as MathML

The Save As MathML option saves the current equation as as MathML markup. When you select this menu option, you will be presented with a panel reviewing the current MathML Export Options. You can either choose to edit these options, or to accept them and continue with the Save.

The most important MathML export option is whether WebEQ Editor should save equations as content MathML markup or presentation MathML markup. However, there are a number of other important preferences that may need to be set, depending on the intended target applications. See the section on Web Math Formats in the full WebEQ Developers Suite documentation for further details.

It is possible that saving an equation in content markup format will fail. In order to save as content markup, WebEQ Editor must be able to determine the mathematical meaning of the current expression. Therefore, it is important to check the equation syntax before saving as content. See Generating Content Markup.
