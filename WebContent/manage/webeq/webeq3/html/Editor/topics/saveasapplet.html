Saving as HTML Applet

The Save As HTML Applet menu option allows you to save the current equation as an applet code for displaying the equation with WebEQ Viewer Control. When you select this menu option, you will be presented with a panel reviewing the current Applet Export Preferences. You can either choose to edit these options, or to accept them and continue with the Save.

In general, you will need to modify your Applet Export Preferences for each project you work on. The default settings are unlikely to work in any given situation. Consult the WebEQ Developers Suite documentation for more information about setting Viewer Control applet parameters.

Note that an equation saved as an applet cannot be re-opened in WebEQ Editor.
