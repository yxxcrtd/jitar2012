Getting Started with WebEQ Publisher

WebEQ Publisher is a document processing program. It reads in a source file containing math markup and produces a new output file where the math has been processed into a format ready for publication.

The source file is an ordinary HTML file, with MathML or WebTeX math markup interspersed with the text. In the HTML output file, the Publisher replaces the source math markup with processed MathML, applet code or image tags. The HTML output file is then ready to post to the Web or view with a browser. Whenever you need to make a change to your document, you should make the change in the source file, then run the Publisher on it again to produce an updated output file.

To use the Publisher, you will generally need to go through the following steps:

  1. Prepare a source file containing either MathML or WebTeX markup for equations.
  2. Specify the source file and output file.
  3. Set the type of processing you want to do.
  4. Fine tune any relevant processing options using the Options Menu
  5. Click "Process Now" on the main panel, and check the output when the Publisher is finished.

Example

A common way to use the Publisher is to process a source HTML file with embedded WebTeX code to produce an output HTML file with a Viewer Control applet tag for each equation. For example, in an HTML source file, an author might write:

The area of a circle is given by $A = \pi r^2$, 
where <em>r</em> is the radius.

In the HTML output file, the Publisher would write out something like:

The area of a circle is given by
<b><applet codebase="classes" code="webeq3.ViewerControl" height="28" width="87">
<param name="eq" value="A= \pi r^2">
</applet></b>, where <em>r</em> is the radius.

Note that the Publisher writes out the height and width of the applet, since Java applets must specify in advance how large they should be, and fills in the codebase attribute in the applet tag.

As an example of the fine tuning processing options, you might want to change the codebase to a different location by using the Viewer Control Options.
