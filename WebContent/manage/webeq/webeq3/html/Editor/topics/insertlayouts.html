Inserting Layout Templates

The following templates can be inserted from the Insert Layout submenu of the Insert Menu

  • Fraction inserts a fraction template
  • Square Root inserts a square root template
  • Radical inserts an "nth root" template
  • Style Change inserts a style change template. The style change template is just like an expression group, except that it allows you to change any MathML style property. Properties set on the style change template apply to all the template nested within it. See How Properties Work for more details.
  • Text inserts a text template. Within a text template the space bar always inserts true spaces and not inivisible multiplication markers. Hardcoded linebreaks are always enclosed in text templates.
  • Expression Group inserts a grouping template. An expression group is an invisible template that serves to collect subexpressions together. Characters like parentheses stretch to the height of the group they are in, so a typical use of an expression group is in creating parenthesized expressions.
  • Parenthesized Expression inserts a parenthesized expressions template. This is just an expression group with two parentheses pre-entered.
