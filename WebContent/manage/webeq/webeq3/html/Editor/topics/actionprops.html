Action Properties Dialog

There are three kinds of action templates. The properties shown in the Action Properties Dialog differ depending on which of type of action template is selected.

Toggle Action Properties

Status line cue for first expression
Determines the text that will be displayed by the Web browser when the reader moves the mouse over the first toggle expression. The cue will be displayed in the WebEQ Editor status line as a preview when the first toggle expression is active.
Status line cue for second expression
Determines the cue text for the second toggle expression in a way exactly analogous to the first status line cue.
Show active expression only/Show both expressions
Switches the editing mode for toggle templates. When both expressions are show, they are displayed in the editor stacked vertically, and grouped with a brace.
toggle active expression
Clicking this button simulates the effect a reader will see when clicking on a toggle expression in a Web browser. Depending on the current editing mode, either the currently displayed expression will switch, or when both are displayed, the alternate expression will be grayed out.

Statusline, Highlight and Link Action Properties

URL
Determines the URL the browser will jump to when a reader clicks on the link expression. This URL must be an absolute URL when equations are to be displayed with the WebEQ Viewer Control.
Status line cue
Determines the cue text that will appear in the browser status line when the reader moves the mouse over the link expression.
Mouse over foreground/background highlight color
Determines the foreground and background colors that will be used for the link expression when the reader moves the mouse over it. This gives a "highlight" effect which makes the link more prominent.

Mouseover Action Properties

The mouseover action properties are exactly analogous to the corresponding properties for the link action template.
