Specifying the Equation Format

WebEQ Publisher generates equation output in four formats. You specify the format you want by selecting from the pulldown menu in the Output Document panel at the bottom of the Publisher main window.

For each equation format, you can set a number of additional processing options from the dialogs accessible from the Options menu. The settings on the Appearance Options panel apply to all output formats. The settings on the other option panels only apply for some of the output formats.

The possible equation output formats are:

Viewer Control Applets
This option generates a WebEQ Viewer Control applet code in the output document. When this option is selected, you will typically need to fine-tune processing settings on the Viewer Control Options panel.
Applets with fallback images
This option generates a WebEQ Viewer Control applet code in the output document just as the previous option. However, it also generates fallback image files, which display in browsers that have Java turned off. When this equation format is selected additional processing setting from both the Translation Options and Image Options panels apply.
Strict MathML
This option writes out processed MathML code for the equations. To view the resulting output, readers will need a MathML capable browser, or a browser with a MathML plug-in. MathML plug-ins and MathML capable browsers usually require additional declarations which must be added to document headers. To specify what kind of additional declarations to include, you will want to adjust the processing settings on the Translation Options panel.
Images Only
This option generates image files for equations. When this equation format is selected, the adjacent "Image" pulldown menu becomes active and determines the graphics format, PNG or JPG, for the images to be generated. PNG is generally superior, and is the default. The HTML output will have the necessary <IMG> tags for the page to display all of the equations properly. The image files themselves will be saved to the local disk. You specify the location on the local disk from the Image Options panel.
