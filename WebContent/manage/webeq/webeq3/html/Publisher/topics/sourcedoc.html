The Source Document

The main window of WebEQ Publisher is divided into two panels. You specify where the source markup will come from in the top panel.

Setting up the Source

The Source Document panel at the top of the main window contains the following input fields:

Math Encoding
By default, the Publisher will automatically detect MathML and WebTeX equations in a document. By changing the encoding, you can specify that MathML only or WebTeX only be processed.
Input from File
When checked, the adjacent text field becomes active, allowing you to enter a filename for the source document to process.
Input from Screen
When checked, Publisher input must be entered directly into the adjacent text area. Note, however, that equation markup entered in this way must still be surrounded by equation delimiters, e.g. $..$ or <math>...</math>
